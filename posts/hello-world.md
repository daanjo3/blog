This is the first page of my blog using a self-written static site generator processing markdown. This project is inspired by [Veit's blog](https://blog.veitheller.de/) and it will therefore probably bear a lot of similarity, at least initially. Though over time I certainly hope I'll be able to give it some flair of my own.

I'm not exactly sure yet what I'll be doing with this project and how long it'll keep me entertained. I guess we'll find out.

### AI disclaimer

First and foremost you can expect that I'll never use AI for the writing of this project. I like the excercise of writing so I won't outsource that.

For the code however, it's gonna be a bit of a mix. I'm a terrible frontend-developer so all styling is probably going to be AI. And some troubleshooting is also definitely going to be AI-assisted. Though the core concepts of this project, the what and the how will be me. But I'll definitely leave navigating the quirks of specific libraries over to an agent. And it might not be a bad idea to let some tooling check whether I didn't forget some words in sentences, though a "normal" spellchecker would also suffice for that.
